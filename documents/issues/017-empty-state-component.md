# フロントエンド: 空状態コンポーネントの改善

## 目的
- 監査ログ一覧の空状態表示を視覚的に強化し、UX を向上させる。

## 背景
- 現在は「ログがありません」というテキストのみ表示
- アイコン付きの空状態コンポーネントがあると視覚的に分かりやすい

## 作業タスク
- [ ] 共通の空状態コンポーネントを作成（アイコン + メッセージ + サブテキスト）
- [ ] `admin_audit_logs/panel.rs` の空状態表示を新コンポーネントに置換
- [ ] 他のページ（例: 申請一覧）でも同様のコンポーネントを適用

## 受入条件
- 空状態がアイコン付きで視覚的に表示される
- 適切なメッセージとサブテキストが含まれる

## テスト
- 目視確認（フィルタで該当なしの状態を作成して表示を確認）

## メモ
- 参考デザイン: Tailwind UI の Empty States
