# 監査ログ: 表示件数の選択

## 目的
- 監査ログ一覧画面で1ページあたりの表示件数をユーザーが選択できるようにする。

## 背景
- 現在は固定20件
- ユーザーによっては一度に多くのログを確認したい場合がある

## 作業タスク
- [ ] `view_model.rs` に `per_page` シグナルを追加
- [ ] `panel.rs` に表示件数セレクトボックスを追加（10, 20, 50件）
- [ ] APIリクエストに `per_page` パラメータを渡す

## 受入条件
- 表示件数が選択可能（10, 20, 50件）
- 選択した件数でページネーションが正しく動作

## テスト
- 目視確認（表示件数を変更してログ一覧が更新されることを確認）
- ページネーションが正しく計算されることを確認

## メモ
- バックエンドは既に `per_page` パラメータに対応済み（MAX_PER_PAGE=100）
